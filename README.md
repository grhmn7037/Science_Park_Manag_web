# نظام إدارة حدائق العلوم والتكنولوجيا - جامعة الكتاب
### Science & Technology Park Management System

نظام متكامل مبني باستخدام **Flask** و **SQLAlchemy** لإدارة طلبات المبتكرين، طرح المشكلات التقنية، وتسجيل حضور السمينار.

**المميزات:**
- لوحة تحكم إدارية محمية بكلمة سر.
- نظام فرز ذكي للمشاريع (أفكار، مشكلات، مشاريع).
- إحصائيات فورية ورسوم بيانية باستخدام Chart.js.
- إمكانية التواصل المباشر مع المبتكرين عبر WhatsApp و Telegram.

**التشغيل:**
1. `pip install -r requirements.txt`
2. `python app.py`
